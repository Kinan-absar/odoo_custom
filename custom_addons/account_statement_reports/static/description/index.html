Account Statement Reports

Customer and Vendor Financial Statements for Odoo with opening balance, running balance, PDF reports, and Excel export.

Overview

This module adds professional account statements to Odoo Accounting, allowing you to generate customer and vendor statements with clear running balances over any selected period.

Key Features

  • Customer Account Statements
  • Vendor Account Statements
  • Opening Balance & Running Balance
  • Clean and Professional PDF Reports
  • Excel Export
  • Easy-to-use Wizards
  • Integrated into Accounting → Reporting
