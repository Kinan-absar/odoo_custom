Send to Sign Purchase Orders

Professional Send-to-Sign workflow for Purchase Orders in Odoo. Automatically control when Purchase Orders are sent for signature and track revisions clearly.

How It Works

  • The Send to Sign button appears only after Purchase Order approval.
  • Once sent for signature, the button automatically disappears.
  • If any change occurs (price, quantity, line added/removed), the button reappears.
  • A new revision number is generated and displayed on the Purchase Order.
  • The updated Purchase Order will trigger the button to appear again automatically.

Key Features

  • Approval-based Send to Sign button
  • Automatic revision tracking
  • Signature state visibility
  • Prevents signing outdated Purchase Orders
  • Fully integrated with Odoo Sign
  • Odoo 18 compatible
