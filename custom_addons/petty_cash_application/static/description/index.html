Petty Cash Management

Structured petty cash control with approvals, VAT, and accounting integration

Accounting

Overview

Petty Cash Management is a complete internal accounting solution designed to control petty cash expenses with clarity, approval workflows, VAT handling, and seamless journal entry creation in Odoo.

This module is ideal for companies that require structured petty cash reporting, category-based expense accounting, and accountant-level validation before posting.

Key Features

  • Petty Cash Reports with automatic sequence numbers
  • Multi-line expense entries with descriptions and invoice/receipt references
  • Category-based expense account mapping
  • Optional VAT per line with automatic VAT calculation
  • Configurable Petty Cash Account, Input VAT Account, and Journal
  • Approval workflow:
    • Draft → Submitted (Users)
    • Submitted → Approved / Refused (Accountants)
    • Approved → Reset to Draft (if needed)
  • Draft Journal Entry generation with correct debit/credit logic
  • Attachment support (receipts, invoices, documents)
  • Excel Import Wizard with template download
  • Professional PDF report for petty cash submissions
  • Chatter integration for communication and tracking
  • Separate security groups for Users and Accountants
